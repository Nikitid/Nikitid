## Projects

### [IKEv2 Manager for OpenWrt](https://github.com/Nikitid/ikev2-manager-openwrt)

LuCI application for outbound and inbound IKEv2 tunnels, domain-based policy routing, per-device rules, managed DNS, and fail-closed routing on OpenWrt.

### [IKEv2 Manager for Ubuntu](https://github.com/Nikitid/ikev2-manager-ubuntu)

Interactive Bash manager for deploying and maintaining a strongSwan IKEv2/IPsec server with ACME certificates, VPN users, client bundles, firewall rules, and diagnostics.

### [MKV Auto Merger](https://github.com/Nikitid/mkv-auto-merger)

Python tool that prepares tracker-style anime and TV folders for Jellyfin, matching external dubbed audio and producing a clean season structure.
